# Developer Portfolio

GitHub Pages로 바로 배포할 수 있는 정적 개발자 포트폴리오입니다.

## 수정할 곳

- `index.html`: 이름, 소개 문구, 프로젝트, 이메일, GitHub, LinkedIn 링크를 수정합니다.
- `styles.css`: 색상, 간격, 카드 스타일을 조정합니다.
- `assets/hero-workspace.png`: 첫 화면 대표 이미지입니다.

## GitHub Pages 배포

1. 이 폴더를 GitHub 저장소로 올립니다.
2. GitHub 저장소의 `Settings`로 이동합니다.
3. `Pages` 메뉴에서 `Deploy from a branch`를 선택합니다.
4. `main` 브랜치와 `/root` 폴더를 선택한 뒤 저장합니다.

빌드 과정이 없는 정적 사이트라서 `index.html`이 저장소 루트에 있으면 바로 배포됩니다.
